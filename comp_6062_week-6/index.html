Week 1

This is the first week of the COMP 6062 course. In this week, we will cover the basics of web development, including HTML, CSS, and JavaScript. You will learn how to create a simple web page and style it using CSS. Additionally, we will introduce you to JavaScript for adding interactivity to your web pages.

By the end of this week, you should be able to create a basic web page with text, images, and links, and apply styles to it using CSS. You will also learn how to use JavaScript to manipulate the DOM and respond to user events.
